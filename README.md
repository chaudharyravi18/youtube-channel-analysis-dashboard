# youtube-channel-analysis-dashboard
#PROJECT OVERVIEW
This project is a comprehensive **Data Analysis Dashboard** built using Power BI to track and analyze YouTube channel performance metrics. Created this interactive dashboard to analyze key metrics like Views, comments, Likes, watch time and Engagement.
#KEY FEATURES & Functionalities
**Interactive KPIs**  dynamic cards: Real-time tracking of Total Views, Likes, Watch Time, comments and enganement rate.
Average Engagement: Custom measures to calculate the average performance per video.
Content Insights: Table with Data Bars to identify top-performing videos.
Engagement Metrics: Donut chart for category-wise likes distribution.
**Interactive Data Bars:** Visual representation of views within the table for quick comparison.
**Donut Chart:** Break-down of **Likes by Category** (Gaming, Comedy, Education, etc.) to understand audience preference.
**Cross-Filtering:** Clicking on any visual updates the entire dashboard dynamically.

#TECHNICAL STACK
**Data Tool:** Microsoft Power BI Desktop.
**Data Source:** Excel / YouTube API Raw Data.

#Project Structure
* `YouTube_Analysis.pbix`: The main Power BI project file.
* `YouTube_Data.xlsx`: The raw dataset used for analysis.
* `powerbi_analysis_dashboard.pfd`: the pfd of main power BI project file.
* `Screenshots/`: Contains images of the final dashboard.
